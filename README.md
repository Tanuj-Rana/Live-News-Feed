# Live News Feed 

A dynamic web application that delivers real-time news updates and allows users to filter content by their interests.

## Features
-  **Real-Time Updates:** Stay up to date with the latest news.
-  **Search & Filter:** Easily find news by category or keyword.
-  **Responsive Design:** Works seamlessly on desktop, tablet, and mobile.

## Technologies Used
- HTML, CSS, and JavaScript
- (Gnews Api)
